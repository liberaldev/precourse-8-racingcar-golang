# Go로 만드는 자동차 경주
프리코스 과제 중 가장 인상깊었던 자동차 경주 과제를 Go 언어로 구현해보는 미션이다. racingcar 등의 명령어를 통해 실행되는 CLI 애플리케이션을 목표로 한다.
## `main.go`
- [x] 옵션 없는 인자는 자동차 이름, `--count` 인자는 횟수로 간주하도록 인자 읽어들어기
- [ ] `cars.go`에 있는 `Cars`구조체의 `sort` 메소드를 이용해 우승자를 선정하는 메소드 구현
- [ ] 우승자 안내 문구 출력
## `cars.go`
- [ ] 차 이름들로 이루어진 배열을 인자로 하여 `Cars` 구조체의 `cars` 필드를 초기화하는 메소드 구현.
- [ ] 각 배열을 순회할때 0에서 9 사이에서 무작위 값을 구한 후 무작위 값이 4 이상의 경우 `steps`의 수를 증가하는 메소드 구현.
- [ ] `steps`의 수를 따라 정렬하는 메소드 구현
- [ ] 얼마나 주행했는지를 출력하는 메소드 구현
